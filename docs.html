BoxCraft Launcher

Un lanzador no oficial de Minecraft Bedrock Edition para sistemas GNU/Linux. Escrito en Python con Qt6 (PySide6), diseñado para ofrecer una experiencia nativa, rápida y con gestión de contenido integrada.

Arquitectura

Basado en mcpelauncher-client para una ejecución estable en Linux.

Aviso Legal

No asociado ni aprobado por Mojang AB o Microsoft Corporation.

01. Instalación del Juego

1

Soporte APK

BoxCraft permite instalar el juego utilizando un archivo APK. Importante: El usuario es responsable de obtener su propio archivo APK legalmente.

2

Extracción de Datos

Utiliza los binarios de mcpelauncher-extract para procesar los datos del juego automáticamente.

03. Compilar para Desarrollo

Para generar el binario ejecutable, utiliza el script de construcción incluido:

python build.py --clean --simple --test

Una vez finalizado, mueve el archivo generado en la carpeta dist/ a la raíz del proyecto para su ejecución.

04. Compilar Paquete .DEB

Si deseas distribuir BoxCraft en distribuciones basadas en Debian/Ubuntu:

1. Crear registro de cambios:

dch --create -D stable --package "boxcraft-launcher" --newversion=1.x.x "New release."

2. Instalar dependencias de construcción:

sudo apt build-dep .

3. Generar paquete:

dpkg-buildpackage -Zxz -rfakeroot -b
